## Deskripsi
Aplikasi sistem informasi pendaftaran berbasis Laravel
yang dibuat untuk memenuhi UAS Pemrograman Web.

## Fitur
- Form pendaftaran
- CRUD data pendaftaran
- Database MySQL
- Laravel MVC

## Cara Menjalankan
1. Clone repository
2. Jalankan `composer install`
3. Copy `.env.example` ke `.env`
4. Atur database
5. Jalankan `php artisan serve`
